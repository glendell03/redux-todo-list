## Tech Stack
- Node jS
- Typescript
- Expo (React Native)
- Nativewind (TailwindCSS)
- Redux Toolkit
- Redux Persist
- Async Storage
- Secure Storage

## Setup
1. Clone the repo
```console
  git clone git@github.com:glendell03/redux-todo-list.git
```
2. Change directory to oo-parking-lot
```console
  cd redux-todo-list
```
3. Install dependencies
```console
  yarn
```
4. Run the development server
```console
yarn start 
```
